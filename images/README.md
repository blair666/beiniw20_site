# Images Folder

## How to Add Your Images:

### 1. Profile Photo/Icon
- **File name**: `profile.jpg` or `profile.png`
- **Size**: 400x400 pixels (square)
- **Format**: JPG or PNG
- **Usage**: Your profile photo in the hero section

### 2. Background Image (Optional)
- **File name**: `hero-background.jpg`
- **Size**: 1920x1080 pixels (landscape)
- **Format**: JPG (for smaller file size)
- **Usage**: Hero section background

### 3. Favicon (Optional)
- **File name**: `favicon.ico`
- **Size**: 32x32 pixels
- **Format**: ICO or PNG
- **Usage**: Browser tab icon

## Current Setup:
- Purple gradient background (your favorite color!)
- Placeholder profile image
- You can replace with your own images by:
  1. Adding your images to this folder
  2. Updating the paths in index.html

## Example Paths:
```html
<!-- Profile Image -->
<img src="images/profile.jpg" alt="Blair W">

<!-- Background Image -->
style="background-image: url('images/hero-background.jpg')"

<!-- Favicon -->
<link rel="icon" href="images/favicon.ico">
```
